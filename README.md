# CalmCraft - Healing Through Hobbies

A gentle, emotion-centered web application that helps users discover hobbies for emotional healing and self-connection.

## 🌸 About

CalmCraft is designed to feel like a soft exhale after a long day. It helps users find gentle hobbies based on their current mood, energy level, and preferences. The platform prioritizes emotional safety, comfort, and self-connection over productivity or achievement.

## ✨ Features

- **Emotion-based hobby recommendations** - Find activities that match how you're feeling
- **Gentle quiz system** - Optional questions with skip functionality
- **"I don't know how I feel" option** - Special support for unclear emotional states
- **My Calm Space** - Private dashboard for saved hobbies
- **Mobile-first responsive design** - Accessible on all devices
- **Local storage** - No accounts needed, privacy-focused

## 🛠 Tech Stack

- **Vue 3** (Composition API)
- **Vue Router** for navigation
- **Tailwind CSS** for styling
- **Vite** for development and building
- **LocalStorage** for data persistence

## 🚀 Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. Navigate to the project directory:
   ```bash
   cd CalmCraft
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:3000`

### Building for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## 🎨 Design Philosophy

- **Emotional safety first** - Users should feel safe within 5 seconds
- **Gentle interactions** - Soft animations, calm colors, supportive language
- **No pressure** - Skip options, flexible timing, no judgment
- **Process over product** - Focus on how activities feel, not outcomes
